# github-triage

Local, read-only GitHub issue triage reports for open-source maintainers.

`github-triage` is being built as a command-line assistant that reads GitHub issues and
produces reviewable Markdown and JSON recommendations for labels, duplicates, missing
details, draft replies, and security-sensitive escalation.

The v0.1.0 MVP is intentionally read-only. It will not post comments, apply labels, close
issues, or otherwise mutate GitHub state.
